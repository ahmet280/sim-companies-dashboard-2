=== SIM COMPANIES DASHBOARD - CHROME EXTENSION ===

INSTALLATION STEPS:
─────────────────
1. Open Chrome
2. Type the following into the address bar: chrome://extensions
3. Toggle the “Developer mode” switch ON in the top-right corner
4. Click the “Load unpacked extension” button
5. Select this folder (sim-extension) → OK

USAGE:
─────────
1. Go to simcompanies.com and log in
2. Click the S icon on the Chrome toolbar
3. Click the “READ DATA” button
4. Wait 2-3 seconds → All data will load!

NOTE:
────
• The Sim Companies tab must be open
• You must be logged in
• You can refresh the data using “↻ REFRESH”
• You can switch between light and dark themes using ☀/☾
